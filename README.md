# Power BI Sales & Profit Analysis

## Overview
This project presents an interactive Power BI dashboard for analyzing sales performance, profit, and profitability across products, customers, regions, and time.

## Dataset
The dataset includes sales transactions with information on products, customers, regions, dates, net sales, and profit.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Features
- KPI cards (Net Sales, Profit, Profit Margin)
- Interactive slicers (Year, Month, Product, Customer)
- Regional and product-level performance analysis
- Monthly and yearly trend analysis
- Detailed tables for drill-down insights

## Key Insights
- A small number of products contribute the majority of total profit
- Profit margins vary significantly across regions
- Certain months show strong seasonal performance
